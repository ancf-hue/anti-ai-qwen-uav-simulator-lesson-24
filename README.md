# 🕵️ Урок 24: Шпион — Стелс-разведка

**БПЛА-тренажёр на Three.js**  
*Часть образовательной платформы [Anti-AI Qwen Academy](https://anti-ai-qwen-academy.vercel.app/)*

---

## 📋 Описание проекта

**«Шпион»** — это бесплатный социальный офлайн-тренажёр для управления беспилотным летательным аппаратом (БПЛА) в стелс-режиме. Игрок управляет квадрокоптером, проникает на вражескую территорию, собирает секретные документы и избегает обнаружения радарами.

Проект разработан в рамках образовательной программы **Anti-AI Qwen Academy** для развития навыков операторов БПЛА, пространственного мышления и тактического планирования.

---

## 🎯 Цель миссии

- 🔍 Проникнуть на вражескую территорию
- 📄 Собрать все секретные документы
- 👁️ Избежать обнаружения радарами
- 🕵️ Использовать стелс-режим для невидимости

---

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `W` / `↑` | Движение вперёд |
| `S` / `↓` | Движение назад |
| `A` / `←` | Движение влево |
| `D` / `→` | Движение вправо |
| `Q` / `Shift` | Набор высоты |
| `E` / `Ctrl` | Снижение |
| `SPACE` | Стелс-режим (невидимость) |
| `ЛКМ + движение мыши` | Обзор 360° |

## 🕵️ Особенности

### Стелс-режим
- 🌫️ Делает дрон невидимым для радаров
- ⚡ Расходует энергию (100%)
- 🐢 Снижает скорость движения
- 🔄 Энергия восстанавливается при выключении

### Система обнаружения
- 📡 Радары вращаются и сканируют территорию
- 🔴 Конус обнаружения показывает зону видимости
- ⚠️ При 3 обнаружениях миссия провалена
- 🎯 Чем ближе к радару — тем выше шанс обнаружения

### Детализированная модель дрона
- 🚁 Реалистичный квадрокоптер с 4 моторами
- 💡 Передние зелёные и задние красные LED-индикаторы
- 📷 Подвесная камера с объективом
- 🌀 Анимированные трёхлопастные пропеллеры
- 🛡️ Защитные ободы вокруг пропеллеров
